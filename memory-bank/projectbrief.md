# Project Brief

## 概要

**Find the Meebit** は、美術館風ワールドに大量の Meebits（VRM アバター）がいる中から、指定されたターゲットを見つけて `E` でインタラクトする 3D 探しゲーム。

## コア体験

- ステージ開始時にターゲットの Meebit 番号とプレビュー画像を覚える
- WASD / モバイルジョイスティックで美術館内を移動
- 近づいて `E` で会話・検査。正解ならクリア
- ステージが進むと NPC 数・ターゲット数が増加
- タイムアタックモードあり（180 秒制限）

## 対象プラットフォーム

| 区分 | 条件 |
|------|------|
| PC | 幅 1024px 以上（Tailwind `lg:`） |
| タッチ UI | 幅 1023px 以下（スマホ **およびタブレット** を同一扱い） |

## ワールド

- 正方形のギャラリー（フレーム約 34m、地面タイル 100×100）
- 入口は +Z 側（プレイヤー開始: `[0, 0, 62]`）
- 奥は -Z 側
- オブジェクト: ゴールデンツリー、ベンチ、モノクロ彫刻、壁パネル、看板、ステージ

## クリエイター NPC

- ID: `npc-shawn-t-art`
- Meebit #11143（Shawn T. Art）
- 専用セリフは `npcGeneration.ts` にハードコード

## 非目標

- サーバーサイドのゲームロジック・マルチプレイ・永続ランキングはない
- DB なし。ローカルストレージでプレイヤー位置のみ保存
